Användarvillkor

I nedanstående text benämns företaget SLDB som "vi". Med "SLDB", "sldb.se", "sidan, eller "tjänsten" menas sldb.se.

1. Lagar

Alla användare av tjänsten åtar sig att följa svensk lag. Vissa lagar kan uppfattas som mer aktuella, exempelvis lagen om upphovsrätt, narkotikastrafflagen, samt brottsbalken (uppvigling, hets mot folkgrupp, barnpornografibrott, olaga våldskildring, olaga hot, ofredande, ärekränkning och pornografisk text och bild). På Rixlex.se kan du läsa mer om lagarna. Om vi får kännedom om grova otillåtna inlägg på sldb.se kan anmälan göras till internetleverantören som då kan varna och även stänga ner internetebonnemanget. Om vi eller någon användare gör polisanmälan kan åtal väckas. Brott mot någon av dessa lagar kan leda till olika typer av straff (se Rixlex.se). Böter eller fängelse är de normala bestraffningarna beroende på brottets art. Observera att det inte krävs ett lagbrott för att vi ska kunna ingripa med varning eller avstängning av kontot.

2. Rättigheter

På sldb.se äger användarna allt material som laddas upp, men genom att ladda upp det ges SLDB tillåtelse att använda, förändra, kopiera och spara allt uppladdat material när, hur och var som helst för all framtid och. Du har ingen rätt att använda sldb.se i eget kommersiellt syfte, som t.ex. försäljning, genomförande av undersökningar eller reklam för någon produkt/företag eller tjänst. Vid sådan förseelse kan vi yrka på skadestånd och ditt konto kan komma att termineras.


SLDB förbehåller sig rätten att avsluta en användares konto när som helst och av vilken anledning som helst.

3. Ändrade villkor

Vi har alltid rätt att ändra de allmänna villkoren på sldb.se om detta behövs. Vi uppmanar användarna att gå in på Användarvillkor för att se om eventuella ändringar skett. Om ändringar är av sådan karaktär att användandet av sldb förändras på ett betydande sätt kommer detta meddelas till användarna. Genom att fortsätta använda dayviews.com anses användaren ha godkänt förändrade regler och villkor. Om användare vill avsluta medlemskapet har användaren möjlighet att avregistrera sig från sldb.se. Detta görs under Inställningar.
